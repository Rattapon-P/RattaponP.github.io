# Rattapon's Portfolio

Personal portfolio website showcasing my work as a Full-stack AI Developer.

## Live Demo

[rattapon-p.github.io/RattaponP.github.io](https://rattapon-p.github.io/RattaponP.github.io/)

## Tech Stack

- HTML5 / CSS3 / JavaScript
- Responsive Design
- Dark/Light Theme Toggle
- CSS Animations & Intersection Observer API

## Features

- **Responsive Design** - Works on all devices
- **Dark/Light Mode** - System preference detection with manual toggle
- **Smooth Animations** - Scroll-triggered animations and typing effect
- **Interactive Stats** - Animated counters

## Sections

- **About** - Background and expertise
- **Skills** - AI/ML, Backend, Frontend, Cloud & DevOps
- **Projects** - Featured work including:
  - Voice Expense Tracker
  - Text to SQL Chatbot
  - Topic Classification
  - Preprocess Text Analysis
- **Experience** - Work history and education
- **Contact** - Email, LinkedIn, GitHub
